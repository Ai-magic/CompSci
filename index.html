HTML Can't Do Computations


Have you ever tried to add 2 and to in HTML? You may say that "I have, in the script tag!!!" Well in the script tag, you program in JavaScript.
HTML just says to the computer "Hey, This is JavaScript code." AKA it provides a structure that says inside this is JavaScript code.

Programming languages have functional purposes. HTML, as a markup language doesn’t really “do” anything in the sense that a programming language does.
HTML contains no programming logic. It doesn’t have common conditional statements such as If/Else. It can’t evaluate expressions or do any math.
It doesn’t handle events or carry out tasks. You can’t declare variables and you can’t write functions. It doesn’t modify or manipulate data in any way.
HTML can’t take input and produce output. Think of it this way: you can’t compute the sum of 2 + 2 in HTML; that’s not what it’s for.
This is because HTML is not a programming language.
